# Prometheus

A lightweight wrapper for efficient data storage and retrieval using Parquet format.

## Features

- **Store files in Parquet format** - Efficiently serialize and store data in the columnar Parquet format
- **Read/query files** - Query/retrieve/write data stored in Prometheus
- **AWS S3 integration** (coming soon) - Future support for connecting to AWS S3 service for cloud storage

## Installation

```bash
pip install -e .
```
